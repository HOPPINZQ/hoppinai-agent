# s08: Background Tasks（后台任务系统）

`s01 > s02 > s03 > s04 > s05 > s06 > s07 > [ s08 ] | s09 > s10 > s11 > s12`

> *"在后台运行慢操作；智能体持续思考"* — 守护线程运行命令，完成后注入通知

## 问题

s07 的 `bash` 工具是阻塞的：模型调用 bash → 等待命令完成 → 返回结果 → 继续思考。如果命令需要几分钟（`npm install`、`pytest`、`docker build`），模型就闲置等待。用户说"安装依赖，同时帮我创建配置文件"，智能体只能顺序执行，无法并行。

**核心矛盾：**
- ❌ bash 阻塞主线程，模型在等待时无法思考
- ❌ 无法并行执行多个耗时任务
- ❌ 长时间命令导致对话超时或体验差

## 解决方案

引入 **BackgroundManager** — 一个基于守护线程池 + 通知队列的后台任务管理器。模型调用 `background_run` 启动任务后**立即返回**，任务在后台线程运行。完成后结果进入 `ConcurrentLinkedQueue` 通知队列，在**下次 LLM 调用前自动注入**到对话中。

```
Main thread                         Background threads
+-------------------+               +-------------------+
| agent loop        |               |                   |
|                   |   spawn       | thread-1: bash    |
| background_run --+------>+------->| npm install       |
| (returns immediately)             | ...               |
|                   |               |        done!      |
| [other tools]    |               | enqueue(result) --+---> notificationQueue
| [LLM call] <-----+------- drain  |                   |     (ConcurrentLinkedQueue)
|  ^inject results |       queue   +-------------------+
+-------------------+               +-------------------+
                                     thread-2: bash
                                     | pytest
                                     | ...              |
                                     | enqueue(result) -+---> notificationQueue
                                     +-------------------+

Timeline:
t0: background_run("npm install")     → 立即返回 "task 00000001 started"
t0: background_run("pytest")          → 立即返回 "task 00000002 started"
t0: write_file("config.yaml", ...)    → 同步执行，完成
t1: [下次 LLM 调用前]
    drainNotifications()              → 注入 task_00000001 和 task_00000002 的结果
    LLM 看到结果 → 继续思考下一步
```

## 工作原理

### 1. BackgroundManager — 后台任务管理器

```java
public class BackgroundManager {
    private final Map<String, BackgroundTaskInput> tasks;          // ConcurrentHashMap
    private final Queue<BackgroundNotification> notificationQueue; // ConcurrentLinkedQueue
    private final ExecutorService executor;                        // CachedThreadPool
    private final AtomicInteger taskIdCounter;                     // 8位十六进制ID

    public BackgroundManager() {
        this.tasks = new ConcurrentHashMap<>();
        this.notificationQueue = new ConcurrentLinkedQueue<>();
        this.executor = Executors.newCachedThreadPool(r -> {
            Thread t = new Thread(r, "bg-task-" + r.hashCode());
            t.setDaemon(true);
            return t;
        });
        this.taskIdCounter = new AtomicInteger(0);
    }
}
```

**核心方法：**

```java
// 启动后台任务 — 立即返回，不阻塞
public String runInBackground(String command) {
    String taskId = generateTaskId();  // 8位十六进制: 00000001
    BackgroundTaskInput task = new BackgroundTaskInput();
    task.setTaskId(taskId);
    task.setCommand(command);
    task.setStatus("running");
    task.setStartTime(LocalDateTime.now().toString());
    tasks.put(taskId, task);
    executor.submit(() -> executeTask(task));
    return String.format("后台任务 %s 已启动，命令: %s", taskId, command);
}

// 执行任务 — 带超时和OS适配
private void executeTask(BackgroundTaskInput task) {
    String os = System.getProperty("os.name").toLowerCase();
    ProcessBuilder pb;
    if (os.contains("win")) {
        pb = new ProcessBuilder("cmd.exe", "/c", task.getCommand());
    } else {
        pb = new ProcessBuilder("bash", "-c", task.getCommand());
    }
    pb.directory(new File(AIConstants.ROOT));
    pb.redirectErrorStream(false);
    Process process = pb.start();

    boolean finished = process.waitFor(300, TimeUnit.SECONDS);
    if (!finished) {
        process.destroyForcibly();
        task.setStatus("timeout");
    } else {
        task.setStatus(process.exitValue() == 0 ? "completed" : "error");
    }
    // 结果截断到 50000 字符
    task.setResult(truncateResult(output, 50000));
    tasks.put(task.getTaskId(), task);
    enqueueNotification(task);
}

// 生成任务ID — 8位十六进制
private String generateTaskId() {
    return String.format("%08x", taskIdCounter.incrementAndGet() & 0xFFFFFFFFL);
}
```

### 2. 通知队列 — 结果注入

```java
// 排空通知队列 — 每次LLM调用前调用
public List<BackgroundNotification> drainNotifications() {
    List<BackgroundNotification> notifications = new ArrayList<>();
    BackgroundNotification notif;
    while ((notif = notificationQueue.poll()) != null) {
        notifications.add(notif);
    }
    return notifications;
}

// 静态注入方法 — 在 chatMessage 中调用
public static void injectBackgroundNotifications(
        List<MessageParam> messageParams,
        BackgroundManager backgroundManager) {
    List<BackgroundNotification> notifications = backgroundManager.drainNotifications();
    if (!notifications.isEmpty()) {
        String notifText = notifications.stream()
            .map(n -> String.format("[后台任务:%s] %s: %s",
                n.getTaskId(), getStatusChinese(n.getStatus()), n.getResult()))
            .collect(Collectors.joining("\n"));
        messageParams.add(MessageParam.builder().role(MessageParam.Role.USER)
            .content("<后台任务结果>\n" + notifText + "\n</后台任务结果>").build());
        messageParams.add(MessageParam.builder().role(MessageParam.Role.ASSISTANT)
            .content("已记录后台任务结果。").build());
    }
}
```

### 4. BackgroundTaskInput — 任务数据模型

```java
public class BackgroundTaskInput {
    private String taskId;    // 8位十六进制: "00000001"
    private String command;   // 原始命令
    private String status;    // running, completed, timeout, error
    private String result;    // 截断到 50000 字符
    private String startTime; // 启动时间

    public String getDisplayString() {
        String statusChinese;
        switch (status) {
            case "running": statusChinese = "运行中"; break;
            case "completed": statusChinese = "已完成"; break;
            case "timeout": statusChinese = "超时"; break;
            case "error": statusChinese = "错误"; break;
            default: statusChinese = status;
        }
        return String.format("[%s] %s: %s", statusChinese, taskId, command);
    }
}
```

### 7. 资源保护

```java
// 自动清理 — 保留最近 100 个任务
public void cleanupOldTasks() {
    if (tasks.size() > 100) {
        List<BackgroundTaskInput> sorted = tasks.values().stream()
            .filter(t -> !"running".equals(t.getStatus()))
            .sorted(Comparator.comparing(BackgroundTaskInput::getStartTime))
            .collect(Collectors.toList());
        int removeCount = tasks.size() - 100;
        for (int i = 0; i < removeCount && i < sorted.size(); i++) {
            tasks.remove(sorted.get(i).getTaskId());
        }
    }
}

// 优雅关闭 — 5秒宽限期
public void shutdown() {
    executor.shutdown();
    if (!executor.awaitTermination(5, TimeUnit.SECONDS)) {
        executor.shutdownNow();
    }
}
```
